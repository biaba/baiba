The Spring Security application with JWT to create stateless session handling.

Used technologies and tools:
- Postgres DB (configured to run locally)
- Jenkins integration
- SonarQube integration (with JaCoCo testing coverage tool)
- Liquibase

Tests include:
- unit tests
- integration tests
- Selenium tests
- Cucumber tests

The application allows users to register/login/logout.
For loged in users the list of cars from DB is displayed. 

The front-end and database functionality is basic with main focus on Spring Security, JWT and session handling.
